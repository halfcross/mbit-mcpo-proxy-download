MCPBehindIT (macOS)

MCPBehindIT is a desktop app that runs the mcpo MCP gateway in Docker and helps you manage MCP server definitions, imports, logs, and settings. You need Docker Desktop running when you use the gateway.

After install, the app may ask for your MindBehind license key (online validation). There is no password on the installer itself.

One-line install (recommended)

Run in Terminal (downloads the latest .dmg from halfcross/mbit-mcpo-proxy releases): 

curl -fsSL https://halfcross.github.io/mbit-mcpo-proxy-download/install.sh | bash

To use another fork, set MBIT_GITHUB_REPO, e.g. MBIT_GITHUB_REPO=myorg/mbit-mcpo-proxy before bash.

The script downloads the latest .dmg from that repo’s GitHub Releases, copies MCPBehindIT.app to /Applications, and runs xattr -cr so macOS does not block the app with a misleading “damaged” message after a browser download.

Manual install

  1. Open the Releases page of mbit-mcpo-proxy.
  2. Download the .dmg for your version.
  3. Open the DMG and drag MCPBehindIT into Applications.
  4. In Terminal, clear quarantine (adjust path if needed): 
    xattr -cr "/Applications/MCPBehindIT.app"
  5. Launch the app from Applications or Spotlight.

Security note

Builds may be ad-hoc signed and are not Apple-notarized. Clearing quarantine is normal for in-house distribution. For stricter deployment, use a Developer ID certificate and notarization in your build pipeline.
