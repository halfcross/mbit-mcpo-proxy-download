Install MCPBehindIT

macOS · Requires Docker Desktop when you use the app · License key is entered inside the app

Terminal — copy and paste
curl -fsSL https://halfcross.github.io/mbit-mcpo-proxy-download/install.sh | bash

Select the line above, paste into Terminal, press Return.

Manual install (download DMG yourself)

Download the latest .dmg from this repo’s Releases. Open it, drag MCPBehindIT to Applications, then run: 

xattr -cr "/Applications/MCPBehindIT.app"

Ad-hoc signed builds may show a “damaged” warning until quarantine is cleared. Notarized builds are not distributed here.
